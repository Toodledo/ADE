Color Picker Directive

A directive that allows you to pick a color from color picker popup.

A directive allows you to pick a color from color picker popup. It has a predefined palette of colors. A maximum of 20 colors can be specified.

The directive will broadcast a message when the edit starts and when it finishes so that your controllers and models can handle the edit properly.
