Decimal filter

A filter that displays a number as a decimal with a specified precision. This is a wrapper for Angular's built in filter so that we can display invalid inputs better.

  • Decimal 1234.3453: {{ 1234.3453 | decimal:2 }}
  • Decimal string "1234.3453": {{ "1234.3453" | decimal:2 }}
  • Invalid decimal "decimal": {{ "integer" | decimal }}

Decimal editing directive

A directive that makes a decimal editable.

The directive will broadcast a message when the edit starts and when it finishes so that your controllers and models can handle the edit properly.
